GAFFERMESH

The Decentralized Compute Market for Football Stadiums.

When 60,000 fans pack a stadium, cellular networks crash. GafferMesh allows fans to purchase AI tactical analysis completely off-grid by routing requests to local devices in the stands.

Built on the Tether Tri-Stack

Winning the Developers Cup by utilizing every layer of the decentralized ecosystem.

Pears P2P Network

Zero cloud servers. We use Hyperswarm and Hypercore to create a local device-to-device mesh network directly in the stadium stands, bypassing dead cellular networks entirely.

WDK Agent Wallets

Fans use self-custodial Tether wallets. Smart agents automatically sign gasless micro-payments of USDt to incentivize users with high-power laptops to provide compute power.

QVAC Local AI

All tactical inference runs completely on-device using the QVAC SDK. Football logic is processed locally without sending sensitive fan data or predictions to cloud APIs.
